# 🎓 Academic Performance Predictor

An end-to-end machine learning project that predicts student academic performance using multiple regression models, data analysis, and visualization techniques.

---

## 🚀 Project Overview

This project aims to predict students' final scores based on key factors such as:

- Study hours  
- Attendance  
- Sleep hours  
- Previous academic performance  

The system applies machine learning techniques to analyze patterns and generate accurate predictions.

---

## 🧠 Features

- Data generation and preprocessing  
- Multiple model training (Linear Regression, Decision Tree, Random Forest)  
- Model comparison using evaluation metrics  
- Visualization of results  
- Feature importance analysis  

---

## 📊 Models Used

- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  

---

## 📈 Evaluation Metrics

- Mean Absolute Error (MAE)  
- Root Mean Squared Error (RMSE)  
- R² Score  

---

## 🛠️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib
